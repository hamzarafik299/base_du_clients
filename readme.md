# Service Manager

Une application Full Stack développée avec FastAPI, SQLite et JavaScript.

## Fonctionnalités

- Gestion des contacts (CRUD)
- Ajouter un contact
- Modifier un contact
- Supprimer un contact
- Interface HTML/CSS/JavaScript
- API REST avec FastAPI

## Technologies

- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- HTML
- CSS
- JavaScript

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

Ouvrir `frontend` avec Live Server.
